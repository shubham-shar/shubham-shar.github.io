Neighbourhood Map Project

This project consists of making a Neighborhood Map webpage showing maps of particular area. Then it lets user to search for a particular place as well can search from a specific place given in the list or markers, which is just below the search bar.
It lets user to show all markers by clicking show markers button or hide them all by clicking the hide markers button.
It lets user to change to night mode and back to normal on a single click on Night mode On/Off button Lastly, it lets user to get wikipedia articles related to the place or string that the user will enter in the search bar.

Arcade Game Project

ARCADE GAME
This game is very simple to play. To play just open `index.html` in any browser. Then player control the hero with the arrow key. Points are earned on the bases of the number of times the hero reaches the water without touching the bugs in the way.
if the player touches the bug then the points Earned will be resetted.
